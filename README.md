# Sublime Nag Screen Removal

This removes the annoying nag screen that appears every several saves inside of sublime.

**There is a 99% chance this breaks if Sublime updates because it directly references the addresses to patch**

## Usage

If you want to compile it yourself, run:

```shell
gcc SublimePatcher.c -o SublimePatcher.exe
```

If you don't have gcc then too bad, figure it out yourself.

Or you can just check releases for the exe file if you trust me.
Which you shouldnt.
Don't trust anyone.

When you want to use the exe:

1. Drag and drop `SublimePatcher.exe` inside of the file where `sublime_text.exe` is located:
2. Right click -> Run as Administrator
3. Have fun, or don't, you have free will
